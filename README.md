# Movie Website

## Project Description
This movie website allows users to search for and explore movies using data from the TMDB API. Users can view details about movies, including cast, reviews, and ratings.

## Features
- Search for movies by title.
- View detailed information about each movie.
- Browse popular and trending movies.
- User-friendly interface with responsive design.

## Setup Instructions
1. Clone the repository:
   ```
   git clone https://github.com/sfnasrooooo/rosa.git
   ```
2. Navigate to the project directory:
   ```
   cd rosa
   ```
3. Install the required dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```

## TMDB API Key Configuration
1. Sign up at [TMDB](https://www.themoviedb.org/) and get your API key.
2. Create a `.env` file in the root of the project and add the following line:
   ```
   REACT_APP_TMDB_API_KEY=your_api_key_here
   ```
3. Replace `your_api_key_here` with your actual TMDB API key.
